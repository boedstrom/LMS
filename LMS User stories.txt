User stories:

Som generell användare:
A1	vill jag kunna logga in	så att jag kan använda systemet
	1	Jag startar applikationen
	2	Login-sidan kommer upp
	3	Jag skriver in användar-id och lösenord i respektive fält och klickar på OK
	4	En översiktsvy vars innehåll är beroende på användarens behörighet visas


Som student:
B1	vill jag kunna se vilken kurs jag går
	1	Min aktuella kurs visas i den första översiktsvyn

B2	vill jag kunna se vilka kurskamrater jag har
	1	Jag klickar på länken "Klassen" till höger om kurs-listan
	2	En lista över vilka som går på min kurs visas.
	3	För att att komma tillbaka till översiktsvyn för kurser klickar jag på länken "Tillbaka till Kurser"

B3	vill jag kunna se vilka moduler som ingår i kursen
	1	Jag klickar på länken "Moduler" till höger om kurs-listan
	2	En lista över vilka moduler som ingår i min kurs visas
	3	För att att komma tillbaka till översiktsvyn för kurser klickar jag på länken "Tillbaka till Kurser"

B4	vill jag kunna se vilka aktiviteter som ingår i modulerna
	1	Jag klickar på länken "Moduler" till höger om kurs-listan
	2	En lista över vilka moduler som ingår i min kurs visas
	3	Jag klickar på länken "Aktiviteter" till höger om den aktuella modulen
	4	En lista över vilka aktiviteter som ingår i den modulen visas
	5	För att att komma tillbaka till översiktsvyn för moduler klickar jag på länken "Tillbaka till Moduler"
	6	För att att komma tillbaka till översiktsvyn för kurser klickar jag på länken "Tillbaka till Kurser"

B5	vill jag kunna ladda ner dokument
	1	Jag klickar mig fram till den sida ("Kurs", "Modul", "Aktivitet") där dokumenten finns.
	2	Jag klickar på dokumentets länk, ändrar eventuellt katalog att spara dokumenten i och trycker "OK".

B6	vill jag kunna ladda upp inlämningsuppgifter
	1	Jag klickar på mig fram till aktuell aktivitet via "Kurs"-, "Modul"- och "Aktivitet"-vyerna.
	2	Jag klickar på "ladda upp"-länken vid den aktuella aktiviteten
	3	En dialog visas där jag kan välja det dokument jag vill ladda upp
	4	Jag klickar på "OK" för att ladda upp filen
